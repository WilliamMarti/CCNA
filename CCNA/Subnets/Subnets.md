###### Classful Subnets

|     | First Binary | First Oct Range | Prefix Length |
| --- | ------------ | --------------- | ------------- |
| A   | 0xxxxxxx     | 0-127           | /8            |
| B   | 10xxxxxx     | 128-191         | /16           |
| C   | 110xxxxx     | 192-223         | /24           |
| D   | 1110xxxx     | 224-239         |               |
| E   | 1111xxxx     | 240-255         |               |

A/B/C - Can be assigned to a host

D/E - Special
